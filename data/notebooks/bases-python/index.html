Bases en Python

Le langage Python est un langage de programmation très populaire dans le domaine de la Data en raison de sa simplicité, de sa flexibilité et de la richesse de ses bibliothèques.
C'est un langage de haut niveau qui permet aux data scientists de se concentrer sur l'analyse des données plutôt que sur les détails de la programmation.
Une fondation solide en langage Python est essentielle pour le Datascienstist.

Les variables

Les variables sont des conteneurs qui stockent des valeurs. En Python, vous pouvez créer une variable en lui assignant une valeur à l'aide de l'opérateur d'affectation (=). Par exemple :


        x = 10

Les types de données

Python prend en charge plusieurs types de données, notamment les entiers (int), les nombres à virgule flottante (float), les chaînes de caractères (str), les listes (list), les tuples (tuple), les ensembles (set) et les dictionnaires (dict). Par exemple :


        name = "Alice"  # str
        age = 30        # int
        height = 1.75   # float
        hobbies = ["reading", "hiking"]  # list

Les structures de contrôle (boucles, conditions)

Les structures de contrôle permettent de diriger le flux d'exécution du programme. Les boucles (for, while) sont utilisées pour répéter des blocs de code, tandis que les conditions (if, elif, else) permettent de prendre des décisions en fonction de certaines conditions. Par exemple :


        # Boucle for
        for i in range(5):
            print(i)

        # Condition if
        if age > 18:
            print("Vous êtes majeur.")
        else:
            print("Vous êtes mineur.")

Les fonctions

Les fonctions sont des blocs de code réutilisables qui effectuent une tâche spécifique. En Python, vous pouvez définir une fonction en utilisant le mot-clé `def`. Par exemple :


        def greet(name):
            print(f"Hello, {name}!")

        greet("Alice")

Les classes

Les classes permettent de créer des objets personnalisés en définissant un plan pour leur structure et leur comportement. En Python, vous pouvez définir une classe en utilisant le mot-clé `class`. Par exemple :


        class Person:
            def __init__(self, name, age):
                self.name = name
                self.age = age

            def introduce(self):
                print(f"Je m'appelle {self.name} et j'ai {self.age} ans.")

Les modules

Les modules sont des fichiers contenant du code Python qui peuvent être importés dans d'autres scripts. Ils permettent de structurer le code et de le réutiliser. Par exemple :


        import math
        print(math.pi)

Maîtriser les bibliothèques de base

Apprenez à utiliser les bibliothèques essentielles pour la datascience, telles que :

  • NumPy pour les calculs numériques
  • Pandas pour la manipulation des données
  • Matplotlib et Seaborn pour la visualisation des données

NumPy pour les calculs numériques

NumPy est une bibliothèque fondamentale pour la manipulation de tableaux et les calculs numériques en Python. Elle offre des fonctionnalités puissantes pour travailler avec des données multidimensionnelles. Par exemple :


        import numpy as np
        # Créer un tableau NumPy
        arr = np.array([1, 2, 3, 4, 5])

Pandas pour la manipulation des données

Pandas est une bibliothèque de manipulation de données qui fournit des structures de données flexibles et des outils pour travailler avec des données tabulaires. Par exemple :


        import pandas as pd
        # Créer un DataFrame Pandas
        data = {'Name': ['Alice', 'Bob', 'Charlie'], 'Age': [25, 30, 35]}
        df = pd.DataFrame(data)

Matplotlib et Seaborn pour la visualisation des données

Matplotlib est une bibliothèque de visualisation de données qui permet de créer des graphiques statiques, animés et interactifs. Seaborn est une bibliothèque basée sur Matplotlib qui offre des graphiques statistiques plus attrayants et faciles à utiliser. Par exemple :


        import matplotlib.pyplot as plt
        import seaborn as sns
        # Visualiser les données avec Matplotlib
        plt.hist(df['Age'])
        plt.show()

        # Visualiser les données avec Seaborn
        sns.barplot(x='Name', y='Age', data=df)
        plt.show()

Apprendre les outils de développement

Familiarisez-vous avec les outils de développement tels que :

  • Jupyter Notebook pour l'exploration interactive des données
  • Anaconda pour la gestion des environnements et des packages
  • Git et GitHub pour faciliter votre travail en datascience

Jupyter Notebook pour l'exploration interactive des données

Jupyter Notebook est un environnement de développement interactif qui permet de créer et de partager des documents contenant du code, des visualisations et du texte. Il est largement utilisé en datascience pour l'exploration et l'analyse des données. Par exemple :


        # Vous pouvez exécuter du code Python dans une cellule Jupyter Notebook
        print("Hello, Jupyter!")

Anaconda pour la gestion des environnements et des packages

Anaconda est une distribution Python qui facilite la gestion des environnements et des packages. Il inclut des outils tels que conda pour créer des environnements virtuels et installer des bibliothèques. Par exemple :


        # Créer un environnement conda
        conda create -n myenv python=3.8
        # Activer l'environnement
        conda activate myenv
        # Installer une bibliothèque
        conda install numpy

Git et GitHub pour faciliter votre travail en datascience

Git est un système de gestion de versions décentralisé qui vous permet de suivre les modifications de votre code et de collaborer avec d'autres développeurs. GitHub est une plateforme en ligne qui héberge des dépôts Git et offre des fonctionnalités supplémentaires pour la collaboration et la gestion de projets. Par exemple :


        # Initialiser un dépôt Git
        git init
        # Ajouter des fichiers au suivi
        git add .
        # Valider les modifications
        git commit -m "Initial commit"
        # Pousser les modifications vers GitHub
        git push origin main
